Для решения данной задачи я создал изображение эталона obj.jpg, которое получил "вручную" в графическом редакторе (грубо говоря предположил что у нас есть фронтальное изображение предмета). Это, наверное, немного спорный шаг.
Затем стандартным SIFTом ищем и сопоставляем особые точки эталона с остальными изображениями (для удобства итерации картинки названы 1,2..5) и обратным преобразованием матрицы гомографии получаем изображения со сцены "пиведенные" в исходному. Они сохраняются в отдельную папку.
После этого каждый пиксель выходного изображения - это минимум по этому же пикселю среди всех (пробовал медиану - некоторые блики остаются).
